# Governance

Bek is open-source infrastructure for agent teammates.

## Core Commitments

- Trust primitives stay in open core: access bundles, approvals, audit, basic budgets, and memory scoping.
- The default UX remains one visible Slack teammate.
- Provider-specific integrations remain adapters.
- Public APIs and SDK contracts require an RFC before breaking changes.

## Decision Process

- Small changes: maintainer review.
- Public API or policy model changes: RFC.
- License, governance, or open-core boundary changes: public RFC and maintainer approval.
