# Trademarks

Bek, the Bek name, and the `@bek` teammate identity are project marks of the Bek contributors unless a separate trademark owner is published later.

## Allowed Uses

You may use the Bek name to:

- refer truthfully to the open-source project,
- describe compatibility with Bek,
- discuss, review, or compare Bek,
- identify an unmodified copy of the project.

## Please Do Not

- imply your fork, service, connector, hosted deployment, or commercial offering is endorsed by the Bek maintainers unless it is,
- use a confusingly similar name for a competing agent teammate or hosted service,
- use the Bek name in a way that suggests users are interacting with the official project when they are not,
- remove license or notice files from redistributed copies.

## Forks And Integrations

Forks and integrations should make their relationship to Bek clear. Suggested phrasing:

```txt
Built with Bek
Compatible with Bek
Forked from Bek
```

Avoid phrasing that suggests official status:

```txt
Official Bek Cloud
Bek Certified
Bek Enterprise
```
